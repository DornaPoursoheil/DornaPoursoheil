## Aktuelles Projekt

### KPI Confidence Scoring System

End-to-End-Datenprojekt zur Verarbeitung und Analyse von Verkehrsdaten mit:

- **Python**
- **PostgreSQL**
- **n8n**
- **Streamlit**
- **Power BI**

Ziel des Projekts ist die Berechnung und Überwachung von KPIs inklusive eines **Confidence Scores** zur Bewertung der Datenqualität auf Basis von Data Quality Checks und Anomalie-Erkennung.
